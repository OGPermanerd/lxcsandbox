## Milestone Summary

**Key Decisions:**
- Make credential copying default (no opt-in flag needed)
- Add GitHub/GitLab to known_hosts via ssh-keyscan (doesn't rely on host having them)
- Copy credentials to both root and dev users for consistency

**Issues Resolved:**
- Claude Code auth error requiring manual /login in new containers
